# Bamboo Flags Website (Production)

Production build output for the Bamboo Flags company website, deployed via GitHub Pages.

## Description

This repository contains the compiled/built static HTML files for the Bamboo Flags website. It is the production deployment version that serves the live website.

## Structure

- `index.html` - Homepage
- `about.html` - About page
- `ai-solutions.html` - AI Solutions services
- `automation.html` - Automation services
- `process-optimization.html` - Process optimization services
- `web-development.html` - Web development services
- `careers.html` - Careers page
- `CNAME` - Custom domain configuration

## Deployment

The site is deployed via GitHub Pages. Changes are automatically published when pushed to the repository.
